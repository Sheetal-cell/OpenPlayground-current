# Mental Health Awareness Informational Website

## 📌 Overview
This project is a simple and responsive awareness website that provides information about common mental health conditions:

- Anxiety
- Depression
- Stress
- Burnout

## ✨ Features
- Clean and calming UI
- Responsive card layout
- Helpful tips button for each condition
- Beginner-friendly structure

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ⚠️ Disclaimer
This website is for awareness and educational purposes only.
It does not provide medical advice or diagnosis.

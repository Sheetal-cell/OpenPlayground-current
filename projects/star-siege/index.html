📖 Dynamic A* Pathfinding Manual

The Campaign:

You must clear 5 sectors. Each sector has a fixed number of enemies. Eliminate all of them to win.

How to Play:

  • Building: Click on the grid to build a tower. Towers cost 50 credits.
  • Blocking: The engine will physically prevent you from placing a tower if it completely blocks the only valid path to the base!
  • Dark Walls: These are permanent environmental obstacles. You cannot build on them.
  • Combat: Towers automatically detect enemies in range and fire.
